# Breadcrumbs

Breadcrumbs is a lightweight browser game built on Wikipedia.

You start on one article and must reach another using only links inside articles.
The goal is to find the shortest possible path.

---

## How to play

1. Open `index.html` in a modern browser
2. Click links inside the article text
3. Reach the target page in as few moves as possible

No accounts, no tracking, no backend.

---

## Tech

- Vanilla HTML, CSS, and JavaScript
- Wikipedia MediaWiki API (`action=parse`)
- Single-file implementation
